# Team Roles

**Project:** Student Study Group Finder

Roles rotate each sprint so every member experiences each responsibility.

## Role Descriptions

- **PM / Scrum Master** — Runs stand-ups, tracks sprint progress, removes blockers, maintains backlog.
- **QA Lead** — Writes test plans, verifies acceptance criteria, reports and tracks bugs.
- **DevOps Lead** — Manages repo, CI/CD pipeline, deployments, and environment configuration.
- **Backend Lead** — Owns server-side architecture, database schema, API design and implementation, authentication/security logic, and backend performance and reliability.
- **UX / Frontend Lead** — Owns UI design decisions, wireframes, accessibility, and frontend code quality.
